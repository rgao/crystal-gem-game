Gems are Outrageous: Collecting Game

You will be given a random number as goal at the start of the game.

Each gem has its own value; clicking on a gem will add its value to your score. The values are initially hidden.

If your score matches the goal, then you win. If your score exceeds the goal, you lose.

Each time the game starts, the value of each gem is randomized.
